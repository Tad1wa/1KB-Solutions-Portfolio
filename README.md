# 1KB-Solutions-Portfolio
1KB Solutions — Portfolio

Personal portfolio of Tadiwa Marimo, a full-stack developer & UI/UX designer based in Harare, Zimbabwe.

Tech Stack

React 19 + TypeScript
Vite 7 — build tool
Tailwind CSS v4 — styling
Framer Motion — animations
Wouter — client-side routing
TanStack Query — data fetching
shadcn/ui — base UI components

Features

Liquid glass UI aesthetic inspired by iOS 26
Custom HUD reticle cursor
Animated page transitions with scan-bar wipe
Boot sequence loading screen
Fully responsive across mobile and desktop
Multi-page routing (Home, About, Skills, Journey, Contact)

Running Locally
bash# From the workspace root
pnpm install
cd artifacts/portfolio
pnpm dev
